# Memory Machine — Giles Project Context

## Signal Dreams
A concept album combining generative TouchDesigner visuals with modular synthesizer music. Core narrative: a machine learning to see — moving from abstract data toward human movement and figurative content. Defined artwork, narrative arc, and traditional album structure.

## Drift
An evolving umbrella project for generative visual and modular synth experiments. Distinct from Signal Dreams. Core philosophy: letting work emerge naturally rather than forcing predetermined frameworks. Space for open-ended exploration without fixed outcomes.
